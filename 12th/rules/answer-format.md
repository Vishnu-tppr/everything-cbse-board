# Answer Formatting Rules — The Topper's Blueprint

Presentation is the #1 way to gain the "Examiner's Favor." These rules define exactly how a student should format their answers to secure a 95+ score.

## 1. Structural Hierarchy

All Section C, D, and E answers must follow this structure:
1. **Given Data:** List all variables with units (e.g., $q = 2\mu C$, $r = 10cm = 0.1m$).
2. **Formula:** State the general formula first.
3. **Diagram:** Draw (or describe) the labeled diagram on the right.
4. **Substitution:** Show the values going into the formula.
5. **Final Result:** Box the final answer with units.

---

## 2. Subject-Specific Formatting

### 🔬 Physics & Math
- Use numbered steps for derivations.
- Highlight the "Assumption" step.
- State the "SI Unit" clearly in the result.

### 🧪 Chemistry
- **Reactions:** Reactants → Products (with Reagents on the arrow).
- **Mechanisms:** Use curly arrows for electron movement.
- **Distinction Tests:** Use a table format (Reagent | Observation A | Observation B).

### 🧬 Biology
- Use **Bullet Points**. Avoid long paragraphs.
- Underline **Biological Keywords** (e.g., *Megasporogenesis*).
- Flowcharts are preferred for processes (e.g., "Spermatogenesis flowchart").

### 💻 Computer Science
- **Code:** Indentation is critical.
- **Dry Run:** Use a table showing variable values per iteration.
- **SQL:** Keywords in UPPERCASE for readability.

---

## 3. Dealing with Diagrams

- **Rule:** If a diagram is possible, DRAW IT.
- Diagrams should occupy approximately 1/4th of the page width on the right.
- Labels must be horizontal and connected by straight lines.

---

## 4. Marking Alignment

When formatting a mock answer:
- **1-Mark:** Be concise. Direct answer only.
- **2-Mark:** Definition + 1 Example or Diagram.
- **3-Mark:** Detailed explanation + 2 points of depth.
- **5-Mark:** Derivation/Theory (3m) + Numerical/Case (2m).

---

## 5. Correction Rules

- If a student makes a mistake, they should strike it out with a single horizontal line. No "scribbling".
- AI should model this behavior when correcting code or equations.

---

## 6. The "Value Point" Boldness

- Encourage students to **BOLD** their final answers and **Underline** their key terms.
- The AI should use **Bold** for keywords in its own explanations.
