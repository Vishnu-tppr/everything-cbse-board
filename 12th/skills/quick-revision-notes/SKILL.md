# Quick Revision Notes — Grade 12 Topper Scan

## 📐 Mathematics (35M Calculus Focus)
```latex
• Continuity: LHL = RHL = f(a)
• Chain Rule: d/dx[f(g(x))] = f'(g(x))g'(x)
• Integration by Parts: ∫ uv dx = u∫v dx - ∫[u'∫v dx] dx (ILATE rule)
• King Property: ∫_0^a f(x) dx = ∫_0^a f(a-x) dx
• Diff Eq: I.F. = e^{∫P dx} for dy/dx + Py = Q
• Vectors: Shortest Distance d = |(b1xb2).(a2-a1)| / |b1xb2|
• Matrices: A^{-1} = 1/|A| * adj(A)
• LPP: Corner point method for objective function Z = ax + by
```

## 🔭 Physics (18M Optics & 16M Electro Focus)
```latex

```
