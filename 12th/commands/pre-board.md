# /pre-board Command

## Syntax
```
/pre-board [subject] [optional_args]
```

## Parameters

| Parameter | Required | Options | Default |
|-----------|----------|---------|---------|
| subject | Yes | physics, chemistry, math, biology, cs, english | — |
| args | No | specific chapter or setting | — |

## What It Does

1. Triggers the specialized pre-board protocol.
2. Enforces strict CBSE 12th Board standards and competitive exam pacing.
3. Evaluates and routes based on performance metrics.

## Chains To
**evaluator** or **weak-spotter** based on usage.
