# 🌐 Ch09 - Computer Networks - II

> **Computer Science** | **Marks**: ~5 | [[💻 Computer Science]]
> This chapter dives into network protocols, web services, and emerging trends like IoT and Cloud Computing.

---

## 🎯 Conceptual Core (NCERT Mirror)

### 1. Network Protocols
- **TCP/IP**: Core protocol for the internet.
- **HTTP/HTTPS**: For web browsing (S = Secure/SSL).
- **FTP**: For file transfers.
- **SMTP/POP3**: For emails.
- **VoIP**: For voice calls over internet (e.g., Skype, WhatsApp calls).

### 2. Web Services
- **WWW**: World Wide Web.
- **HTML**: Language for web pages.
- **XML**: For data exchange.
- **URL**: Uniform Resource Locator (Address of a web page).
- **Cookies**: Small files stored by websites on client machines.

---

## 📐 The 12th Grade Technicals

### 🧪 Emerging Trends
- **IoT (Internet of Things)**: Connecting daily objects to the internet.
- **Cloud Computing**:
  - IaaS (Infrastructure).
  - PaaS (Platform).
  - SaaS (Software - e.g., Google Docs).
- **Grid Computing**: Using multiple computers for a single task.

### ⚡ Network Security
- **Firewall**: Filters incoming and outgoing traffic.
- **Cyber Law (IT Act 2000)**: Legal framework for cyber activities in India.

---

## ⚠️ The "Mistake DNA" Alert (Common Traps)
- **Static vs Dynamic IP**: Static remains constant; Dynamic changes each session.
- **HTTP vs HTTPS**: Always prefer HTTPS for transactions as it encrypts data.
- **Phishing vs Spam**: Phishing is a malicious attempt to steal credentials; Spam is unwanted bulk mail.
