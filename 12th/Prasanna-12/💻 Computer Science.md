## 🚀 The Python Powerhouse (40 Marks)

Python is the core of 12th CS. Focus on:
- **Functions**: Scope (LEGB rule), parameter passing, mutable vs immutable.
- **File Handling**: Text (read/write), Binary (pickle - dump/load), CSV (reader/writer).
- **Data Structures**: Stacks (Push/Pop operations) - 5M guaranteed.
- **Exceptions**: Try-Except blocks.

---

## 🗄️ Databases & SQL (20 Marks)
- **SQL Queries**: Joins, Group By, Having, Aggregate functions.
- **Interface**: Python to MySQL connectivity (`mysql.connector`).

---

## 🌐 Networking (10 Marks)
- **Case Study**: Designing a network layout for a school/office (4M).
- **Protocols**: HTTP, TCP/IP, FTP, SMTP.
- **Devices**: Hub, Switch, Gateway, Router.
