# Answer Writing — Board Exam Craft

## The Examiner's Perspective

CBSE examiners evaluate using **value points** — specific keywords, steps, or concepts that carry marks. Your job: hit every value point.

```
EXAMINER CHECKLIST:
✓ Did the student mention the correct keyword/term?
✓ Is the formula written correctly?
✓ Are steps shown clearly?
✓ Is the diagram drawn and labeled?
✓ Is the final answer clearly stated?
✓ Are units mentioned?
```

## Answer Structure by Mark Value

### 1-Mark (MCQ / Very Short)
- Select the correct option with confidence
- For Assertion-Reasoning: evaluate BOTH independently, then check causation
- Time: 1-1.5 minutes per question

### 2-Mark (Very Short Answer)
```
Structure:
  Formula/Definition (1 mark)
  Application/Example (1 mark)
Length: 2-4 lines
Time: 3-4 minutes
```

### 3-Mark (Short Answer)
```
Structure:
  Point 1 with explanation (1 mark)
  Point 2 with explanation (1 mark)
  Point 3 with explanation OR diagram (1 mark)
Length: 6-8 lines
Time: 5-7 minutes

For numericals: Formula → Substitution → Calculation → Answer with units
```

### 5-Mark (Long Answer)
```
Structure:
  Given: [state all given data]
  To find/prove: [state objective]
  Solution/Proof:
    Step 1 (with reason)
    Step 2 (with reason)
    Step 3...
  Therefore: [final answer / Hence Proved]
  Diagram (if applicable)
Length: 12-15 lines + diagram
Time: 10-12 minutes

For theory: 5 well-explained points with examples
```

### 4-Mark (Case Study)
```
Structure:
  Read passage → Extract data
  (i) Answer sub-part (1 mark) — usually direct
  (ii) Answer sub-part (1 mark) — application
  (iii) Answer sub-part (2 marks) — higher-order thinking
Time: 8-10 minutes
```

## Subject-Specific Answer Patterns

### Mathematics
- Write formula → Substitute → Simplify → Answer with units
- Geometry: Given → To prove → Construction → Proof → QED
- Always box/underline the final answer

### Science
- Physics: Formula → Numbers → Calculation → Unit
- Chemistry: Balanced equation + state symbols + reaction type
- Biology: Diagram first → Explanation → Use biological terms

### Social Science
- History: Date → Event → Cause → Impact (chronological)
- Geography: Location → Conditions → Data → Example
- PolSci: Constitutional provision → Explanation → Example
- Economics: Data → Comparison → Analysis → Conclusion

### English
- Literature: Context → Explanation → Quote → Theme connection
- Writing: Format (1M) → Content (3M) → Expression (1M)
- Grammar: Identify error → Apply rule → Correct sentence

## Keywords That Score Marks

Each subject has specific keywords examiners look for:

```
SCIENCE: "due to", "because of", "results in", "hence", "therefore"
MATH: "Given", "To find", "Solution", "Hence proved", "Therefore"
SST: "According to", "Constitutional provision", "As per Article..."
ENGLISH: "The author conveys", "This suggests", "The theme of"
```

## Common Format Mistakes (Mark Killers)

| Mistake | Impact | Fix |
|---------|--------|-----|
| No Given/To Find in numericals | -1M | Always start with Given |
| Diagram without labels | -1 to -2M | Label every part |
| No units in answer | -0.5M | Always write units |
| "Hence proved" missing | -1M | Always end proofs properly |
| Wrong letter format (English) | -1 to -2M | Memorize format |
| No state symbols in equations | -0.5M | Always (s)(l)(g)(aq) |
| Exceeding word limit | Time waste | Practice word count |
| Illegible handwriting | -1 to -3M | Practice daily writing |

## Learning from Toppers' Answer Sheets

Study actual topper answer sheets (available online from CBSE) and note:
1. **Neatness**: Clear sections, no overwriting
2. **Structure**: Headings, bullet points, numbered steps
3. **Diagrams**: Clean, labeled, with ruler
4. **Completeness**: Every question attempted, every sub-part answered
5. **Key terms underlined**: Final answers, important formulas, definitions
