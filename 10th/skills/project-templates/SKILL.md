# Project Templates — 5/5 in Internal Assessment

## SST Project: Disaster Management (5M)

### Format
```
1. COVER PAGE: Title, Name, Class, Section, Roll No, School Name, Year
2. ACKNOWLEDGEMENT (half page): Thank teacher, parents, sources
3. CERTIFICATE (half page): School stamp, teacher signature
4. INDEX: Chapter-wise page numbers
5. CONTENT (10-15 pages):
   Chapter 1: Introduction to Disaster Management
   Chapter 2: Types of Disasters (Natural + Man-made)
   Chapter 3: Case Study (choose ONE — earthquake/flood/cyclone/fire)
   Chapter 4: Preparedness and Mitigation
   Chapter 5: Role of Government Agencies (NDMA, SDMA)
6. CONCLUSION (1 page)
7. BIBLIOGRAPHY: List all sources

PRESENTATION:
  ✅ Handwritten (most schools require)
  ✅ Diagrams, maps, newspaper clippings
  ✅ Charts or tables showing disaster data
  ✅ Clear headings with underlining
```

### Scoring Criteria
```
Content relevance: 2M
Presentation & neatness: 1M
Research & data: 1M  
Conclusion & learning: 1M
```

## Science Project (5M)

### Suggested Topics
```
1. Study of soil types in your locality
2. Water quality testing (pH, hardness, dissolved oxygen)
3. Biodegradable vs non-biodegradable waste survey
4. Solar energy applications model
5. Study of plant growth under different conditions
```

### Format
```
1. COVER PAGE + ACKNOWLEDGEMENT + CERTIFICATE
2. AIM / HYPOTHESIS
3. MATERIALS & METHODOLOGY
4. OBSERVATIONS (tables, data, photos)
5. ANALYSIS & DISCUSSION
6. CONCLUSION
7. BIBLIOGRAPHY

KEY: Include actual data — measurements, readings, observations
NOT just copied text from internet
```

## English Portfolio (5M)

### Components
```
1. Creative writing samples (2-3 pieces)
   - Poem, short story, or diary entry
2. Reading response journal
   - Reflections on 2-3 books/articles read
3. Letter/article samples
   - 1 formal letter + 1 article (practice pieces)
4. Grammar exercises (completed worksheets)
5. Self-assessment reflection

FORMAT:
  ✅ Organize in a folder with dividers
  ✅ Each piece should be dated
  ✅ Include teacher-marked drafts showing improvement
  ✅ Neat handwriting or clean printouts
```

## Timeline for Projects
```
APRIL-MAY: Choose topic, collect sources
JUNE-JULY: Write content (Chapter 1-3)
AUGUST: Complete writing (Chapter 4-5), add visuals
SEPTEMBER: Final check, binding, submit
                                                     
Don't wait until December — projects done early score higher
because you have time to incorporate teacher feedback.
```
