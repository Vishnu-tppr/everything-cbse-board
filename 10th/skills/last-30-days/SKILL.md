# Last 30 Days — The Countdown Plan

## Overview
```
Day 30-21: RAPID REVISION (complete syllabus scan)
Day 20-11: PRACTICE MODE (mock papers + weak chapters)
Day 10-4:  POLISH (formula drill + diagram drill + keywords)
Day 3-1:   CALM DOWN (light scan + rest + confidence)
```

## Day 30-21: Rapid Revision Phase

### Schedule Template (8 hours/day)
```
Morning (3 hrs): Subject 1 — full chapter-wise revision
  → Scan NCERT + your notes
  → Solve 5 key questions per chapter
  → Review diagrams and formulas

Afternoon (3 hrs): Subject 2 — same format
  → Focus on chapters you scored LOW in mock tests

Evening (2 hrs): Subject 3 — quick revision
  → Only formulas, keywords, and weak areas
  → 1 Oswaal sample paper section
```

### Daily Rotation
```
Day 30: Math + Science (Physics)
Day 29: SST (History + Geography) + English
Day 28: Math + Science (Chemistry)
Day 27: SST (PolSci + Economics) + Tamil
Day 26: Science (Biology) + Math (weak chapters)
Day 25: English + SST (map work)
Day 24: Math + Science (numericals only)
Day 23: Tamil + SST (dates/events/timeline)
Day 22: Science (diagrams only) + Math (formulas)
Day 21: English (writing formats) + FULL REVIEW of weak chapters
```

## Day 20-11: Practice Phase

### Schedule Template (8 hours/day)
```
Morning (3 hrs): 1 FULL MOCK PAPER (timed — strict 3 hours)
Afternoon (2 hrs): Mark + analyze the mock paper
  → Classify every mistake (C/R/P/X/A)
  → Note which chapters/topics lost marks
Evening (3 hrs): Fix weak areas from today's mock
  → Re-study the specific topics
  → Solve 10 similar questions
  → Update weak chapter tracker
```

### Mock Paper Schedule
```
Day 20: Math mock (Oswaal sample paper)
Day 19: Science mock (KV paper)
Day 18: SST mock (Oswaal sample paper)
Day 17: English mock (CBSE sample paper)
Day 16: Tamil mock
Day 15: Math mock (KV paper — harder)
Day 14: Science mock (CBSE sample paper)
Day 13: SST mock (KV paper)
Day 12: English mock (Oswaal)
Day 11: Math mock (Navodaya paper — hardest)
```

## Day 10-4: Polish Phase

```
Day 10: Formula sheet drill — write ALL formulas from memory
Day 9:  Diagram bank drill — draw ALL 27 diagrams from memory
Day 8:  Keyword lists — scan keywords per chapter
Day 7:  Math weak chapter: 20 problems rapid fire
Day 6:  Science weak chapter: 10 questions + diagrams
Day 5:  SST revision: dates, map work, source-based
Day 4:  English: 2 letters + 2 analytical paragraphs + grammar
```

## Day 3-1: Calm Phase

```
Day 3: Light revision (formulas + diagrams ONLY)
  → NO new problems, NO new chapters
  → Pack exam bag
  → Sleep by 10 PM

Day 2: Minimal study (1-2 hours max)
  → Quick scan of formula sheet
  → Visualize: "I walk into the exam, I'm calm, I write well"
  → Meet friends briefly (don't discuss syllabus)
  → Sleep by 9:30 PM

Day 1 (exam day): 
  → Follow board-day-guide exactly
  → Light breakfast, reach early
  → You've prepared for 10 months. Trust it.
```

## Red Flags During Last 30 Days
```
⚠️ If mock scores < 70%: Focus ONLY on weak chapters, skip strong ones
⚠️ If feeling overwhelmed: Take a half-day break, exercise, sleep
⚠️ If running out of time: Prioritize subjects with lowest mock scores
⚠️ If new syllabus topics not covered: Study NCERT text only (skip problems)
```
