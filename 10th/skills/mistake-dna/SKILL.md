# Mistake DNA Tracker

## The Problem with Simple Mistake Tracking

Most trackers say: "You got Electricity wrong." But WHY did you get it wrong matters more:
- Didn't understand the concept? → Re-learn
- Understood but forgot? → Spaced repetition
- Knew it but wrote wrong format? → Answer-writing practice
- Calculation error? → Slow down, double-check
- Couldn't apply to case study? → CBQ practice

## Mistake Categories (DNA Types)

### 🔴 Type C — Conceptual
```
SYMPTOMS: Fundamentally wrong answer, wrong formula used, confused concepts
EXAMPLE: Using V = IR for power instead of P = VI
ROOT CAUSE: Gap in understanding
FIX: Re-read NCERT → watch explanation → solve 10 basic problems
TIMELINE: 2-3 days per concept
```

### 🟡 Type R — Recall
```
SYMPTOMS: "I knew this but forgot", blank during exam, mixed up similar facts
EXAMPLE: Forgot that Gandhi withdrew Non-Cooperation after Chauri Chaura
ROOT CAUSE: Not enough revision, no spaced repetition
FIX: Flashcard + write from memory + revise at intervals (1d, 3d, 7d, 14d)
TIMELINE: Ongoing (daily 15-min recall practice)
```

### 🟠 Type P — Presentation
```
SYMPTOMS: Correct concept but wrong answer format, missing steps, no diagram
EXAMPLE: Wrote correct Ohm's law but didn't draw circuit diagram (lost 1M)
ROOT CAUSE: Doesn't know examiner expectations
FIX: Study topper answer sheets, practice answer-writing drills
TIMELINE: 1 week of focused format practice
```

### 🔵 Type X — Careless (Execution)
```
SYMPTOMS: Knew everything but made silly error in calculation/copying
EXAMPLE: Calculated 7×8=54 instead of 56, copied wrong sign
ROOT CAUSE: Speed pressure, not double-checking
FIX: Slow down on calculations, verify key steps, box final answers
TIMELINE: Build habit over 2-3 weeks
```

### 🟣 Type A — Application (CBQ-specific)
```
SYMPTOMS: Knows concept in isolation but can't apply to new scenario/passage
EXAMPLE: Knows photosynthesis but can't answer "why does a plant in a dark room turn yellow?"
ROOT CAUSE: Learning is rote, not conceptual
FIX: Practice CBQ/case study questions, discuss concepts in real-world context
TIMELINE: 3-4 weeks of CBQ-focused practice
```

## Mistake DNA Logging Format

After every test, log each wrong answer:

```markdown
## Test: [Test Name] — [Date] — [Score: X/Total]

### Wrong Answer 1:
- Question: [brief description]
- Subject/Chapter: [e.g., Science/Electricity]
- Marks Lost: [X]
- My Answer: [what I wrote]
- Correct Answer: [what I should have written]
- Mistake DNA: [C/R/P/X/A]
- Root Cause: [specific reason]
- Fix Action: [specific remediation step]
- Fix Deadline: [date]
- Fixed? [ ]
```
