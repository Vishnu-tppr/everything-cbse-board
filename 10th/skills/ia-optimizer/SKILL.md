# IA-20 Optimizer — The Silent 100 Marks

## Why This Is Critical

```
5 subjects × 20 IA marks each = 100 marks
100 marks that DON'T depend on a 3-hour exam
100 marks where 20/20 is achievable with PLANNING, not talent
100 marks most students lose 5-15 on through carelessness

Target: 100/100 in IA → only need 395/400 in theory for 495/500
```

## IA Breakdown Per Subject (CBSE 2026-27)

### Mathematics (20 marks)
```
Periodic Tests (best 2 of 3):        10 marks
  - PT1 (July-Aug), PT2 (Nov-Dec), PT3 (Jan-Feb)
  - Best 2 averaged
  
Multiple Assessment:                   5 marks
  - Math Lab Activities (practical geometry, graphing, verification)
  - Projects (data collection, statistical analysis, mathematical modeling)
  
Portfolio:                             5 marks
  - Classwork notebooks (neat, complete, corrected)
  - Assignments and worksheets
  - Self-assessment sheets
```

### Science (20 marks)

## IA Action Timeline

```
APRIL-MAY (Year Start):
  □ Buy dedicated notebooks for EACH subject (not shared)
  □ Create project topic list (discuss with teachers)
  □ Start classwork from Day 1 — no backlogs

JUNE-JULY:
  □ Prepare for PT1 (2 weeks before)
  □ Score target: 9+/10 in every PT1
  □ Start Science lab record (don't accumulate experiments)
  □ Start SST Disaster Management project outline

AUGUST-SEPTEMBER:
  □ Complete Science project file
  □ Submit SST project draft for teacher review
  □ Maintain notebook corrections (don't skip any)
  □ English ASL preparation

OCTOBER-NOVEMBER:
  □ Prepare for PT2 (2 weeks before)
  □ Score target: 9+/10 in every PT2
  □ Complete all lab experiments (Science)
  □ Finalize SST project

DECEMBER-JANUARY:
  □ PT3 preparation — this is the LAST chance to improve PT scores
  □ If PT1 was weak, PT3 MUST be strong (best 2 of 3)
  □ Complete all portfolios
  □ Get teacher feedback on notebooks

FEBRUARY (Pre-Board):
  □ IA marks usually finalized by now
  □ Verify with teachers: "Are all my submissions complete?"
  □ Any missing notebook/assignment = lost marks
  □ Fix anything that's incomplete IMMEDIATELY
```

```
Lab/Practical      | 18/20 done | 4/5    | ⚠️ 2 experiments pending
Project            | Submitted  | 5/5    | ✅ Done
Portfolio          | Partial    | 3/5    | 🔴 Notebook incomplete!
────────────────────────────────────────────────
TOTAL ESTIMATE:    | 29/35 → scaled to 17.1/20
GAP TO 20/20:      2.9 marks at risk
ACTION NEEDED:     Complete notebook, ace PT3
```

## Golden Rules for 20/20 IA

1. **Never miss a periodic test** — if you're sick, get a retest
2. **Notebook = marks** — maintain from Day 1, get corrections done
3. **Submit projects early** — teachers give better marks to early submissions
4. **Talk to your teachers** — ask "Is there anything I'm missing for IA?"
5. **PT strategy: best 2 of 3** — if PT1 was bad, PT2+PT3 must be excellent
6. **Lab records: real-time** — do the record the same day as the experiment
7. **No backlogs** — the moment you skip one assignment, it snowballs
