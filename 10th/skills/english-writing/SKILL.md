# English Writing Formats — 5M Each, Format = Free 1M

## 1. Formal Letter (5 marks)

### Format (1M — free if you memorize this)
```
[Sender's Address]
[City, Pin Code]

[Date: DD Month YYYY]

[Receiver's Designation]
[Receiver's Address]
[City, Pin Code]

Subject: [4-5 words summarizing purpose]

Respected Sir/Madam,

[Para 1: Introduction — why you're writing — 2 lines]

[Para 2: Main content — details, facts, request — 4-5 lines]

[Para 3: Conclusion — action expected — 2 lines]

Yours faithfully,
[Name]
```

### Marking: Format(1) + Content(2) + Organization(1) + Accuracy(1) = 5M

### Common Types
```
Letter to Editor: pollution, traffic, civic issue
Letter of Complaint: water supply, road condition, noise
Letter of Enquiry: course details, admission process
Letter to Principal: leave, event permission, facility request
```

## 2. Analytical Paragraph (5 marks)

### Format
```
[NO title, NO byline — just a single paragraph]

The given [chart/graph/table/data] presents information about [topic].
It can be observed that [main trend 1]. Furthermore, [trend 2].
A comparison reveals that [comparison point]. Interestingly,
[surprising finding or exception]. On the other hand, [contrast].
The data clearly indicates that [conclusion/summary].
```

### Marking: Content(2) + Organization(2) + Accuracy(1) = 5M

### Rules
```
✅ Single paragraph (NOT multiple paragraphs)
✅ Formal, objective tone (NO personal opinions — "I think")
✅ Cite specific numbers/data from the chart
✅ Include at least 4 data points/trends
✅ Use connectors: furthermore, moreover, however, on the other hand
❌ Don't start with "In my opinion" or "I believe"
❌ Don't add information not in the data
```

## 3. Article Writing (5 marks)

### Format
```
                    [TITLE — Eye-catching]
                    By: [Your Name]

[Introduction: Hook the reader — question, fact, quote — 2 lines]

[Body Para 1: Causes / Main points — 3-4 lines]

[Body Para 2: Effects / Solutions / Examples — 3-4 lines]

[Conclusion: Call to action or final thought — 2 lines]
```

### Marking: Format(1: title+byline) + Content(2) + Organization(1) + Accuracy(1) = 5M

## 4. Grammar Quick Reference

### Tense Conversion Rules (1M each)
```
Direct → Indirect (Reported Speech):
  Present Simple → Past Simple: "I go" → he said he went
  Present Continuous → Past Continuous: "I am going" → he said he was going
  Past Simple → Past Perfect: "I went" → he said he had gone
  Will → Would: "I will go" → he said he would go
  Can → Could | May → Might | Shall → Should

Time/Place shifts:
  today → that day | tomorrow → the next day
  yesterday → the previous day | here → there
  this → that | now → then | ago → before
```

### Modals (1M each)
```
Can: ability | Could: past ability / polite request
May: permission / possibility | Might: less certain possibility
Must: obligation / strong necessity | Should: advice
Will: future / determination | Would: polite / imaginary
Shall: offer / suggestion (with I/We)
```

## Word Count Rule
```
All writing tasks: 100-120 words
Under 80 words → content marks deducted (too short)
Over 150 words → no penalty BUT wastes time for other questions
Target: exactly 100-110 words = safe zone
```
