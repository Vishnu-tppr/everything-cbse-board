# Diagram Bank — Every Mark-Carrying Diagram

## Why Diagrams = Free Marks
CBSE awards dedicated marks for diagrams in 3M and 5M questions. A correct text answer WITHOUT diagram loses 1 mark. A labeled diagram alone can earn 1-2 marks even if text is weak.

## MUST-KNOW Diagrams (Board Exam Mandatory)

### Biology (12 diagrams)
```
1.  Human digestive system (with labels: mouth, oesophagus, stomach,
    small intestine, large intestine, liver, pancreas)
2.  Human heart (4 chambers, valves, blood vessels, oxygenated/deoxygenated)
3.  Nephron / Excretory unit (Bowman's capsule, glomerulus, tubule, collecting duct)
4.  Human brain (cerebrum, cerebellum, medulla, pons)
5.  Neuron (cell body, dendrite, axon, nerve ending, myelin sheath)
6.  Reflex arc (receptor → sensory nerve → spinal cord → motor nerve → effector)
7.  Human eye (cornea, lens, retina, optic nerve, iris, pupil)
8.  Male reproductive system (testes, vas deferens, urethra, penis, prostate)
9.  Female reproductive system (ovary, fallopian tube, uterus, vagina)
10. Flower structure (sepal, petal, stamen, pistil, anther, stigma, ovary)
11. Stomata (guard cells, chloroplast, stomatal pore, epidermal cells)
12. DNA double helix (simple representation with base pairs)
```

## Diagram Rules for Full Marks
```
✅ Use pencil for diagram, pen for labels
✅ Labels on RIGHT side with horizontal lines
✅ Title below the diagram
✅ Clean lines (use ruler for straight parts)
✅ Minimum 6-8 labels for full marks
✅ Arrows showing direction (blood flow, nerve impulse, light rays)
❌ Never free-hand what should be straight
❌ Never skip labels (unlabeled diagram = half marks)
❌ Never draw too small (use at least ¼ page)
```
