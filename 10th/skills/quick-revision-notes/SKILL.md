# Quick Revision Notes — Last-Minute Scan File

## Mathematics — Key Points Only
```
• Real Numbers: HCF×LCM = product. Euclid's lemma: a=bq+r
• Polynomials: sum=-b/a, product=c/a (quadratic)
• Linear Eq: substitution, elimination, cross-multiplication
• Quadratic: x=(-b±√D)/2a, D=b²-4ac, D≥0 for real roots
• AP: aₙ=a+(n-1)d, Sₙ=n/2[2a+(n-1)d]
• Triangles: AA/SAS/SSS similarity, BPT, Pythagoras
• Coordinate: distance=√[(x₂-x₁)²+(y₂-y₁)²], mid=((x₁+x₂)/2,(y₁+y₂)/2)
• Trig: sin²+cos²=1, 1+tan²=sec², 1+cot²=cosec²
• Circle: tangent⊥radius, tangents from external point are equal
• Mensuration: CSA/TSA/Volume for cylinder, cone, sphere, frustum
• Statistics: mean(3 methods), median=l+[(n/2-cf)/f]×h, mode=l+[(f₁-f₀)/(2f₁-f₀-f₂)]×h
• Probability: P(E)=favourable/total, P(E)+P(Ē)=1
```

## Science — Key Points Only
```
CHEMISTRY:
• Reactions: combination, decomposition, displacement, double displacement, redox
• pH: 0-14, <7=acid, 7=neutral, >7=base
• Metals: reactivity series K>Na>Ca>Mg>Al>Zn>Fe>H>Cu>Ag>Au
• Carbon: tetravalency, catenation, isomers, homologous series (+CH₂)
• STATE SYMBOLS: (s)(l)(g)(aq) — ALWAYS WRITE

PHYSICS:
• V=IR, P=VI=I²R=V²/R, E=Pt, 1kWh=3.6×10⁶J
• Series: R=R₁+R₂, Parallel: 1/R=1/R₁+1/R₂
• Mirror: 1/v+1/u=1/f, m=-v/u (sign convention!)
• Lens: 1/v-1/u=1/f, P=1/f(in m)
• Myopia→concave lens, Hypermetropia→convex lens
• Fleming's Left(motor), Right(generator)

BIOLOGY:
• Nutrition: autotrophic(plants), heterotrophic(animals)
• Respiration: C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + ATP
• Heart: 4 chambers, double circulation (pulmonary+systemic)
• Excretion: kidney→nephron→Bowman's capsule→tubule
• Reproduction: budding, fission, regeneration (asexual); flower parts (sexual)
• Heredity: Mendel's laws, dominant/recessive, XX/XY sex determination
• Evolution: natural selection, speciation, fossil evidence
• Environment: food chain, 10% energy transfer, biodegradable/non-biodegradable
```
