# Marking Scheme Decoded — What Examiners Actually Check

## How CBSE Examining Works
```
1. CBSE releases a "Marking Scheme" to examiners (not public)
2. Each question has specific "value points" — keywords/steps that earn marks
3. Examiners tick value points as they scan your answer
4. No value points found = 0 marks, even if answer is "partially correct"
```

## Value Point System

### 1-Mark Questions (MCQ/VSA)
```
Value points: 1 keyword or correct option
  → Either fully correct (1M) or wrong (0M)
  → No partial marking
  → STRATEGY: Attempt ALL — no negative marking in CBSE
```

### 2-Mark Questions
```
Value points: Formula (0.5M) + Correct answer with units (1.5M)
  OR: Two distinct points (1M each)
  → Partial marks: formula alone = 0.5M, answer without formula = 1M
  → STRATEGY: Always write formula first, then substitute
```

### 3-Mark Questions
```
Value points: 3 distinct points (1M each)
  OR: Definition (1M) + Explanation (1M) + Example/Diagram (1M)
  → Partial marks: any correct point = 1M
  → STRATEGY: Write exactly 3 numbered points. Don't merge them.
```

### 5-Mark Questions
```
Value points split as:
  Math: Given (0.5M) + Formula (0.5M) + Steps (2-3M) + Answer (0.5M) + Diagram (0.5M)
  Science: Definition (1M) + Explanation (2M) + Diagram (1M) + Example (1M)
  SST: 5 distinct points (1M each)
  
  → STRATEGY: 
    Math: ALWAYS show Given/To Find/Formula/Solution/Therefore
    Science: ALWAYS include diagram
    SST: ALWAYS write 5 numbered points (not 3 or 4)
```

### Case Study (4M)
```
Value points:
  Sub-part (i): 1M — usually direct from passage
  Sub-part (ii): 1M — application/analysis
  Sub-part (iii): 2M — deeper analysis OR choose correct option + justify
  
  → STRATEGY: Answer (i) first — it's free marks from the passage
```

## Subject-Specific Examiner Lookouts

### Mathematics
```
Examiners CHECK:
  ✅ "Therefore" / "Hence" statement at the end
  ✅ Units (cm², m³, etc.)
  ✅ Correct sign in answer
  ✅ Complete steps (not jumps)
  
Examiners PENALIZE:
  ❌ Final answer without working → max 1M regardless
  ❌ Wrong formula → 0M for entire calculation
  ❌ Missing units → 0.5M deducted
```

### Science
```
Examiners CHECK:
  ✅ NCERT terminology used
  ✅ Balanced equation with state symbols
  ✅ Labeled diagram
  ✅ Named examples (not just "an acid" — say "hydrochloric acid")

Examiners PENALIZE:
  ❌ Diagram without labels → 0M for diagram portion
  ❌ Equation without state symbols → 0.5M lost
  ❌ Vague language: "it reacts" → WHAT reacts with WHAT?
```

### Social Science
```
Examiners CHECK:
  ✅ Specific names, dates, and events
  ✅ Constitutional article numbers where relevant
  ✅ Data/statistics where relevant

Examiners PENALIZE:
  ❌ Vague answers: "there was a movement" → WHICH movement?
  ❌ Wrong dates → 0M for that point
  ❌ Writing 3 points when 5 are asked → max 3M
```

### English
```
Examiners CHECK:
  ✅ Format compliance (letter format, title+byline for article)
  ✅ Word limit adherence
  ✅ Grammar accuracy
  ✅ Connectors and flow

Examiners PENALIZE:
  ❌ Missing format → 1M lost automatically
  ❌ High error density (3+ errors per sentence) → accuracy mark lost
  ❌ Off-topic content → content marks lost
```
