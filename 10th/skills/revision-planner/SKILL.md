# Revision Planner

## The Three-Pass Revision System

### Pass 1: Deep Revision (8-10 weeks before exam)
- Re-read NCERT chapter by chapter
- Solve all NCERT exercises again
- Make summary notes (1 page per chapter)
- Draw all diagrams from memory
- Time: 4-5 hours/day across subjects

### Pass 2: Practice Revision (4-6 weeks before exam)
- Solve sample papers and previous year papers
- Focus on weak chapters (from tracker)
- Timed practice — simulate exam conditions
- Review mistakes from Pass 1
- Time: 5-6 hours/day

### Pass 3: Exam-Week Revision (Last 7 days before each exam)
- Formula sheets and mind maps ONLY
- Quick chapter summaries (15 min/chapter)
- Solve 1 full paper for the subject exam next day
- Focus on high-weightage chapters
- Time: 3-4 hours (not more — rest is crucial)

## Subject-Wise Daily Time Allocation

### Regular Study Period (6-8 months before exam)
| Subject | Daily Time | Focus |
|---------|-----------|-------|
| Mathematics | 90 min | Problem solving + formula practice |
| Science | 75 min | Concepts + diagrams + equations |
| Social Science | 60 min | Reading + map work + dates |
| English | 45 min | Literature + grammar drills |
| Tamil | 30 min | Grammar + creative writing |
| **Total** | **5 hours** | |

### Intensive Period (2-3 months before exam)
| Subject | Daily Time | Focus |
|---------|-----------|-------|
| Mathematics | 120 min | Sample papers + weak chapters |
| Science | 90 min | Full papers + diagram mastery |
| Social Science | 75 min | Papers + map work daily |
| English | 60 min | Writing practice + literature revision |
| Tamil | 45 min | Papers + essay/letter practice |
| **Total** | **6.5 hours** | |

## Weekly Schedule Template

```
MONDAY:    Math (Algebra) + Science (Physics) + English (Literature)
TUESDAY:   Math (Geometry) + Science (Chemistry) + SST (History)
WEDNESDAY: Math (Trigonometry) + Science (Biology) + SST (Geography)
THURSDAY:  Math (Statistics) + SST (PolSci + Eco) + Tamil
FRIDAY:    Full Practice Paper (rotate subjects weekly)
SATURDAY:  Weak Chapter Focus + Map Work + Grammar
SUNDAY:    Revision of week's work + Self-test + Rest
```
