# Geography Map Work — 5 Free Marks

## Why This Matters

Map work carries **5 marks** in the Social Science paper (out of 80). These are:
- The EASIEST marks on the entire paper
- Pure recall + precision (no analysis needed)
- Lost only through lack of practice or careless marking
- Students who practice 20 maps score 5/5. Those who don't lose 2-4 marks.

## Map Marking Rules (CBSE Standard)

```
DO:
  ✓ Use a sharp pencil for marking
  ✓ Mark the location with a dot or small circle
  ✓ Write the name NEXT TO the mark (not far away)
  ✓ Draw a small line connecting name to mark if needed
  ✓ Use standard map symbols (▲ for mines, ⊕ for plants)
  ✓ Write clearly and legibly
  ✓ Practice on printed outline maps (not freehand)

DON'T:
  ✗ Write the name far from the location
  ✗ Use a pen (pencil is standard for map work)
  ✗ Mark multiple items in the same spot without clarity
  ✗ Forget to attach the map to the answer sheet
  ✗ Leave the map blank even if unsure (mark your best guess)
```

## Daily Map Practice Routine (15 minutes)

```
WEEK 1-2: History map items (7 locations)
  Day 1: Mark all 7 from memory → check → correct
  Day 2: Same drill, time yourself (target: 3 minutes)
  Day 3-5: Mix with previous days

WEEK 3-4: Geography — Minerals + Energy (25+ locations)
  Day 1: Iron ore mines (6 locations)
  Day 2: Coal mines (5 locations)
  Day 3: Oil fields (5 locations)
  Day 4: Nuclear power plants (6 locations)
  Day 5: Review all minerals + energy

WEEK 5-6: Dams + Ports + Airports + Steel Plants
  Day 1: Dams (9 locations)
  Day 2: Sea ports (11 locations)
  Day 3: Airports (7 locations)
  Day 4: Steel plants (7 locations)
  Day 5: Review all

WEEK 7+: Full map drill daily
  Mark ALL items (50+ locations) in 10 minutes
  Target: 100% accuracy in under 8 minutes
```
