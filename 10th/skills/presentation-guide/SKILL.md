# Presentation Guide — Make Examiners WANT to Give You Marks

## Why Presentation Matters
```
Examiner marks 50-100 papers per hour
Clean, organized paper → examiner finds value points easily → full marks
Messy paper → examiner misses value points → marks lost
Same content, different presentation → 2-3 marks difference
```

## Answer Sheet Rules

### Page Setup
```
✅ Leave 4-line margin on left (for examiner's marks)
✅ Write question number clearly on left margin
✅ Leave 2 blank lines between questions
✅ Start each new section on a new page (if possible)
✅ Write on BOTH sides of the page
```

### Handwriting
```
✅ Medium size — not too big (wastes pages) or too small (hard to read)
✅ Consistent slant and spacing
✅ Dark blue or black pen ONLY (not gel pens that smudge)
✅ Use pencil for diagrams, pen for labels
✅ If handwriting is poor → write slightly slower, slightly bigger
```

### Underlining & Highlighting
```
✅ Underline key terms, formulas, and final answers
✅ Box the final numerical answer: | R = 6Ω |
✅ Underline NCERT keywords that match value points
✅ Use ruler for underlining (not freehand)
❌ Don't underline everything — only keywords
❌ Don't use highlighters (not allowed in many centers)
```

### Answering Format by Type

#### 1-Mark (MCQ)
```
Q1. (b) ← just the option letter
```

#### 2-Mark
```
Q12. [Formula]
     [Substitution + Answer with units]
```

#### 3-Mark
```
Q18. (i) [Point 1 — one sentence with keyword underlined]
     (ii) [Point 2 — one sentence with keyword underlined]
     (iii) [Point 3 — one sentence with keyword underlined]
```

#### 5-Mark (Math)
```
Q25. Given: [list all given values]
     To Find: [what is being asked]
     Solution:
       [Formula]
       [Substitution step]
       [Calculation step]
       [Simplification]
     Therefore, [final answer with units]
     
     [Diagram if applicable — pencil, labeled]
```

#### 5-Mark (Science/SST)
```
Q26. [Definition or introduction — 1 line]
     
     (i) [Point 1 with explanation]
     (ii) [Point 2 with explanation]
     (iii) [Point 3 with explanation]
     (iv) [Point 4 with example]
     (v) [Point 5 with example]
     
     [Diagram with labels — if Science]
```

### Diagram Presentation
```
✅ Use pencil for drawing, pen for labels
✅ Minimum size: ¼ page (not tiny)
✅ Labels on RIGHT side, horizontal lines
✅ Title below: "Fig: [Diagram Name]"
✅ Arrows for direction (blood flow, current, light)
✅ Use ruler for straight lines (circuits, ray diagrams)
```

## Speed Writing Tips
```
For students who write slowly:
  1. Practice writing 1 page in 4 minutes (time yourself)
  2. Use abbreviations in rough work, full words in answers
  3. Don't re-write mistakes — cross with one line, continue
  4. Skip 2 lines between Qs (don't skip half page)
  5. Write points, not paragraphs (faster + clearer)
```
