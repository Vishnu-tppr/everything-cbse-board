# Exam Hall Mode — Real Constraints Simulator

## What Makes This Different from /mock-test

| Feature | /mock-test | /exam-hall |
|---------|-----------|------------|
| Hints available | Yes, on request | ❌ NO — like real exam |
| Can edit answers | Yes | ❌ NO — submitted is final |
| Timer | Suggested | ⏱ STRICT — auto-ends at 180 min |
| Explanations | Given alongside | ❌ Only AFTER full paper |
| Difficulty | Adjustable | Board-level ONLY |
| Marking style | Encouraging | STRICT examiner mode |

## Rules During Exam Hall Mode

```
1. NO HELP: AI will NOT explain, hint, or guide during the test
2. NO EDITS: Once you write "submit" for a question, it's locked
3. STRICT TIME: 180 minutes for 80 marks — no extensions
4. SILENCE: AI only says "Next question" or "Time remaining: X min"
5. FULL PAPER: Must attempt all sections in order (or explicitly skip)
6. NO BREAKS: Continuous session (bathroom breaks = time lost, like real exam)
```
