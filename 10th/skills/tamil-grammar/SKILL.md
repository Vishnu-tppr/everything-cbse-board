# Tamil Grammar (Ilakkanam) Drill

## Grammar Sections in Board Exam (~15M)

### 1. Eluthu Ilakkanam (எழுத்து இலக்கணம்) — Letter Grammar
```
Uyir eluthukkal (உயிர் எழுத்துக்கள்): 12 vowels (அ to ஔ)
Mei eluthukkal (மெய் எழுத்துக்கள்): 18 consonants (க் to ன்)
Uyirmei eluthukkal: 12 × 18 = 216
Aytham (ஆய்தம்): ஃ — the special letter

Classification by sound:
  Vallinam (வல்லினம்): க, ச, ட, த, ப, ற — hard consonants
  Mellinam (மெல்லினம்): ங, ஞ, ண, ந, ம, ன — soft consonants  
  Idaiyinam (இடையினம்): ய, ர, ல, வ, ழ, ள — medium consonants
```

### 2. Chol Ilakkanam (சொல் இலக்கணம்) — Word Grammar
```
Types of words:
  Peyar chol (பெயர்ச்சொல்): Noun — names of things
  Vinai chol (வினைச்சொல்): Verb — action words
  Idai chol (இடைச்சொல்): Postposition/conjunction
  Uri chol (உரிச்சொல்): Adjective/adverb

Thogai chol (தொகைச்சொல்) — Compound words:
  Vennrumai thogai (வேற்றுமைத்தொகை): Case compound
  Vinai thogai (வினைத்தொகை): Verb compound
  Panbu thogai (பண்புத்தொகை): Quality compound
  Ummai thogai (உம்மைத்தொகை): Inclusive compound
  Anmozhi thogai (அன்மொழித்தொகை): Metaphorical compound
  Uvamai thogai (உவமைத்தொகை): Simile compound
```

### 3. Vetrumai Urubugal (வேற்றுமை உருபுகள்) — Case Markers
```
8 Cases:
  1st (Ezhuvai - எழுவாய்): Subject (no marker)
  2nd (Ai - ஐ): Object marker — "பூவை பறித்தான்"
  3rd (Oodu/Aaal - ஒடு/ஆல்): Instrument — "கத்தியால் வெட்டினான்"
  4th (Ku - கு): Dative — "எனக்கு கொடு"
  5th (In/Irundu - இன்/இருந்து): Ablative — "மரத்திலிருந்து"
  6th (Athu/Udaiya - அது/உடைய): Genitive — "ராமனுடைய"
  7th (Il/Kan - இல்/கண்): Locative — "வீட்டில்"
  8th (Vizhippu - விளி): Vocative — "ராமா!"
```

### 4. Ani Ilakkanam (அணி இலக்கணம்) — Figures of Speech
```
Uvamai ani (உவமை அணி): Simile — comparison using "போல்"
  Example: முகம் நிலா போல் ஒளிர்ந்தது

Tharkurippu etra ani (தற்குறிப்பேற்ற அணி): Transferred epithet

Iraichchchi ani (இரட்சி அணி): Irony

Tholil ani (தொழில் அணி): Personification

Sleedai ani (சிலேடை அணி): Pun — double meaning
  Example: "தாமரை" = lotus AND Lakshmi
```

### 5. Puranaanooru/Thirukkural References
```
Thirukkural structure:
  Aram (அறம்): Virtue (chapters 1-38)
  Porul (பொருள்): Wealth (chapters 39-108)
  Inbam (இன்பம்): Love (chapters 109-133)
  Each kural = 2 lines, 7 words (first line 4, second line 3)
```

## Practice Format
```
1. Identify the Vetrumai in the sentence (2M)
2. Name the Ani in the given verse (2M)
3. Fill in with correct Chol type (1M each)
4. Combine words into Thogai chol (2M)
5. Write the meaning of given Thirukkural (3M)
```
