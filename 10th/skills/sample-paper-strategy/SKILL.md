# Sample Paper Strategy — How to Extract Maximum Value

## Types of Sample Papers (Ranked by Importance)

```
1. CBSE Official Sample Paper ★★★★★ (MUST DO FIRST)
   → Released by CBSE on cbseacademic.nic.in
   → Exact pattern match for actual board paper
   → Includes official marking scheme
   → Usually 1-2 papers per year

2. KV Pre-Board Papers ★★★★☆ (stress test)
   → Slightly harder than CBSE board
   → More application-based questions
   → If you score 70/80 on KV → expect 75-80 on board

3. Oswaal Sample Papers ★★★★☆ (well calibrated)
   → Close to CBSE difficulty level
   → Comes with detailed solutions + mind maps
   → Chapter-wise PYQ compilation is their best feature

4. Arihant Sample Papers ★★★☆☆ (slightly harder)
   → Good for students already scoring 75+/80
   → Some questions above board level — don't panic if tough

5. School Pre-Board Papers ★★★☆☆ (varies by school)
   → Quality depends on your school
   → Treat as practice, not prediction
```

## The 3-Pass Method for Sample Papers

### Pass 1: Timed Simulation (Day 1)
```
→ Strict 3-hour timer (no pausing)
→ No textbook, no notes, no phone
→ Write on answer sheets (practice handwriting + speed)
→ Stop at exactly 3 hours even if incomplete
→ Score: note raw marks HONESTLY
```

### Pass 2: Analysis (Day 1-2)
```
→ Check every answer against marking scheme
→ For each wrong answer, classify:
  C = Conceptual gap (didn't know the concept)
  R = Recall failure (knew it but forgot)
  P = Presentation (knew it but didn't present properly)
  X = Careless (knew it, silly mistake)
  T = Time (knew it but ran out of time)

→ Calculate: "If I fixed all X and T errors, my score would be ___"
  This is your TRUE potential — work on C and R to close the gap
```

### Pass 3: Targeted Fix (Day 2-3)
```
→ For each C error: re-study that chapter section
→ For each R error: add to spaced repetition queue
→ For each P error: practice answer format from topper-patterns
→ For each X error: add to error-analysis drill list
→ For each T error: practice speed on similar questions
```

## Sample Paper Calendar
```

Week 1: CBSE Official Sample Paper (all subjects)
Week 2: Oswaal Paper Set 1 (all subjects)

Week 5: KV Paper Set 2 (all subjects)

Week 8: Final CBSE sample paper (confidence check)
```
