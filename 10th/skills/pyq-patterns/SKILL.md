# PYQ Pattern Analyzer — What CBSE Repeats Every Year

## The Secret: CBSE Doesn't Repeat Questions, It Repeats PATTERNS

```
2019: "Calculate the resistance when V=12V, I=2A"
2020: "Find the current when V=6V, R=3Ω"
2021: "A circuit has R=5Ω. If V=10V, find I and P"
2022: "Two resistors 3Ω and 6Ω in parallel. Find equivalent R"
→ PATTERN: Ohm's law + circuit calculation appears EVERY year as 3M or 5M
```

## Most Repeated Patterns by Subject

### Mathematics (Patterns that appear EVERY year)
```
GUARANTEED 5M Questions (appears 90%+ years):
  • Quadratic word problem (age/number/speed-distance)
  • AP: Find sum of n terms given conditions
  • Trigonometry identity proof
  • Statistics: Find mean/median using grouped data
  • Mensuration: Combination of solids (cone on cylinder, hemisphere on cylinder)

GUARANTEED 3M Questions:
  • Coordinate geometry: Prove collinear / find area of triangle
  • Pythagoras theorem proof or application
  • Probability: Two dice / cards / coins problem
  • BPT (Basic Proportionality Theorem) proof

GUARANTEED 1M MCQs:
  • HCF/LCM of given numbers
  • Nature of roots using discriminant
  • nth term of AP
  • Distance between two points
  • Standard trig values (sin 60°, cos 45°, etc.)
```

### Science (Patterns that appear EVERY year)
```
GUARANTEED 5M Questions:
  • Draw and explain human heart / double circulation
  • Draw ray diagram (concave/convex lens for given object position)
  • Explain Ohm's law with circuit diagram + V-I graph
  • Compare series and parallel circuits (table format)

GUARANTEED 3M Questions:
  • Balance a chemical equation + identify reaction type
  • Explain any ONE life process (digestion/respiration/excretion)
  • Mendel's experiment: monohybrid cross with ratio
  • Properties of acid/base with examples
  • Difference between AC and DC

GUARANTEED CBQ (appears in Case Study):
  • Circuit calculation from data
  • pH identification from given values
  • Ecological scenario (food chain / ozone)
  • Genetic cross interpretation
```

### Social Science (Patterns that appear EVERY year)
```
History:
  • Source-based: Extract from Gandhiji's writings/speeches
  • "Explain any THREE features of [movement]" (3M or 5M)
  • Timeline/sequence of national movement events

Geography:
  • Map work: Locate dams / iron ore mines / cotton textile centres
  • Compare two soil types / crop types (table format)
  • Resource classification with examples

Economics:
  • Data interpretation: HDI / per capita income table
  • Compare formal and informal credit sources
  • "How has globalisation affected Indian economy?" (5M)

Political Science:
  • "Why is power sharing desirable?" (3M)
  • Compare any two political parties
  • Case study: federal/democratic scenario analysis
```

### English (Patterns that appear EVERY year)
```
GUARANTEED:
  • Formal letter (complaint / editor / enquiry) — 5M
  • Analytical paragraph from chart/graph/data — 5M
  • Extract-based questions from First Flight prose — 4-5M each
  • Grammar: Tense conversion, Reported speech, Modals — 1M each
  • Unseen passage: Factual (10M) + Case-based (10M)
```

## How to Use PYQ Patterns

```
1. If a pattern appears EVERY year → practice 10+ variations of it
2. If a pattern appears 4/5 years → practice 5+ variations
3. Start with GUARANTEED 5M patterns — highest marks-per-hour
4. Use Oswaal PYQ section — questions are already sorted by year
5. Time yourself — if you can't solve a pattern in target time, drill more
```
