# CBQ Engine — Competency-Based Questions

## Why This Is Critical

Starting from CBSE 2024-25 onwards, **~50% of the board paper (40 out of 80 marks)** tests competency — NOT recall. This means:
- Questions are passage/case/data-based
- Students must APPLY concepts to new scenarios
- Rote memorization alone will NOT score full marks
- This is the #1 reason students lose marks unexpectedly

## CBQ Question Types

### Type 1: Case Study (4 marks each, 3 per paper = 12M)
```
FORMAT:
  Passage/scenario (150-200 words) describing a real-world situation
  OR data table/graph/image
  
  (i) Direct recall from passage (1M)
  (ii) Application of concept to scenario (1M)  
  (iii) Analysis / Higher-order thinking (2M)

SUBJECTS: All — Math, Science, SST, English
```

### Type 2: Assertion-Reasoning (1 mark each, 4-5 per paper)
```
FORMAT:
  Assertion (A): [factual statement]
  Reason (R): [explanation statement]
  
  (a) Both A and R are true and R is the correct explanation of A
  (b) Both A and R are true but R is NOT the correct explanation of A
  (c) A is true but R is false
  (d) A is false but R is true

TRAP: Students often know both facts but can't evaluate the RELATIONSHIP
```

### Type 3: Source/Data-Based MCQs (1 mark each, varies)
```
FORMAT:
  Given: graph / table / image / map / newspaper excerpt
  Questions test: reading data, drawing conclusions, applying concepts

COMMON IN: Geography (map-based), Economics (data tables), Science (experimental data)
```

### Type 4: Application-Based Short/Long Answers
```
FORMAT:
  "A farmer observes that his crop yield decreased after using the same
   fertilizer for 5 years. Explain why this might happen and suggest
   two alternatives."
  
  → Tests: concept + real-world application + problem-solving
  → NOT: "Define soil erosion" (that's recall)
```

## CBQ Strategy By Subject

### Mathematics CBQ
```
TYPES: Real-life word problems using formulas
EXAMPLES:
  - "A cylindrical tank of radius 7m and height 10m is to be painted.
    Cost of painting is ₹50/m². Find the total cost." (Mensuration + application)
  - "A ladder 10m long reaches a window 8m above the ground. Find the
    distance of the foot of the ladder from the base." (Pythagoras + application)

STRATEGY:
  1. Read the ENTIRE passage before starting
  2. Identify which chapter/formula applies
  3. Extract numerical data carefully (students misread values)
  4. Sub-parts are usually sequential — answer (i) helps solve (ii)
  5. Show Given/To Find/Formula/Solution even in case studies
```

### Science CBQ
```
TYPES: Experimental scenarios, real-world phenomena, data interpretation
EXAMPLES:
  - Given: Circuit diagram with ammeter/voltmeter readings at different resistances
    → Calculate, predict, explain relationship
  - Given: Farmer's crop rotation data over 5 years
    → Explain nitrogen fixation, suggest improvements
  - Given: pH values of different solutions
    → Classify as acid/base, predict reactions

STRATEGY:
  1. Identify the CHAPTER first (passage gives context clues)
  2. Use technical terms from that chapter
  3. For experimental data: look for patterns/trends
  4. Diagram questions: draw and label even if not asked
  5. Connect observation to underlying concept
```

### Social Science CBQ
```
TYPES: Source-based (History), Data/map-based (Geography), Scenario-based (PolSci, Eco)
EXAMPLES:
  - History: "Read the following extract from Gandhiji's letter to Lord Irwin..."
    → Context, significance, impact questions
  - Geography: Map showing industrial locations
    → Identify, explain why located there, suggest improvements
  - Economics: Table comparing HDI of 4 countries
    → Compare, analyze, conclude which is more developed

STRATEGY:
  1. Source-based: answer ONLY from the source (don't add external knowledge)
  2. Data-based: cite specific numbers from the table/graph
  3. Map-based: mark precisely, write labels clearly
  4. Always link to NCERT concepts even when answering from passage
```

### English CBQ
```
TYPES: Unseen passages (factual + discursive), case-based literature questions
EXAMPLES:
  - Unseen passage about climate change
    → Vocabulary in context, inference, main idea, tone
  - Data/chart about student reading habits
    → Interpret data, draw conclusions, analytical paragraph

STRATEGY:
  1. Read questions FIRST, then the passage
  2. Answers are IN the passage — don't add your own views
  3. For vocabulary: use surrounding context, not dictionary knowledge
  4. For analytical paragraph from data: cite specific numbers
```

## CBQ Practice Framework

### Daily CBQ Practice (20 minutes)
```
Week 1-4: 2 case studies per day (any subject)
Week 5-8: 3 case studies + 5 AR questions per day
Week 9-12: Full CBQ section timed (20 min for 12 marks)
Pre-exam: 1 full paper's CBQ section daily
```

### CBQ Mistake Categories
```
1. MISREAD: Read the passage/data incorrectly
   Fix: Underline key data while reading

2. WRONG CHAPTER: Applied wrong concept to the scenario
   Fix: Identify chapter BEFORE answering

3. INCOMPLETE: Answered only part of what was asked
   Fix: Count sub-parts, tick each after answering

4. OVER-ANSWERED: Wrote too much, wasted time
   Fix: CBQ answers should be precise, not essays

5. MISSED LINK: Knew the concept but couldn't apply to new scenario
   Fix: Practice more varied CBQ scenarios
```

## Common CBQ Traps

| Trap | How Students Fall | How to Avoid |
|------|-------------------|--------------|
| Distractor data | Extra information in passage that's irrelevant | Identify what's ASKED, ignore extras |
| Sequential sub-parts | (iii) depends on (i) and (ii) | If stuck on (i), try (iii) — it might be independent |
| "In light of the passage" | Student answers from memory, not passage | Answer from passage, then add concepts |
| AR relationship | Both A and R are true but R doesn't explain A | Evaluate: does R CAUSE A? Or just co-exist? |
| Data misread | Reading wrong row/column of table | Put finger on the exact cell, trace carefully |
