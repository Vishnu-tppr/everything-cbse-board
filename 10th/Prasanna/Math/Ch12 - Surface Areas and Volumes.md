# Ch12 — Surface Areas and Volumes ⭐

> **Unit VI: Mensuration** | **Marks**: 5-8 | [[📘 Mathematics]]
> ⭐ **HIGH YIELD** — 5M LA likely

## Key Formulas

```
CUBE:       SA = 6a²,           V = a³
CUBOID:     SA = 2(lb+bh+hl),   V = lbh
CYLINDER:   CSA = 2πrh,  TSA = 2πr(r+h),  V = πr²h
CONE:       CSA = πrl,   TSA = πr(r+l),    V = (1/3)πr²h,  l = √(r²+h²)
SPHERE:     SA = 4πr²,          V = (4/3)πr³
HEMISPHERE: CSA = 2πr²,  TSA = 3πr²,       V = (2/3)πr³
FRUSTUM:    CSA = π(r₁+r₂)l,    V = (1/3)πh(r₁²+r₂²+r₁r₂)
            where l = √[h²+(r₁-r₂)²]
```

## Common Exam Traps ⚠️

1. Hemisphere TSA = CSA + base circle area (3πr², NOT 2πr²)
2. Not using frustum formula when a cone is cut
3. Unit conversion errors (cm to m)
4. Forgetting to write units (cm², cm³)
