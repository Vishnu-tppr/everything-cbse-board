# Ch05 — Arithmetic Progressions

> **Unit II: Algebra** | **Marks**: 4-6 | [[📘 Mathematics]]

## Key Concepts

- AP: a, a+d, a+2d, ...
- nth term, sum of n terms
- If Sₙ is given, aₙ = Sₙ - Sₙ₋₁

## Key Formulas

```
nth term: aₙ = a + (n-1)d
Sum: Sₙ = n/2 [2a + (n-1)d] = n/2 [first + last]
Common difference: d = aₙ - aₙ₋₁
Middle term of odd AP: (n+1)/2 th term
```

## Common Exam Traps ⚠️

1. Confusing aₙ (nth term) with Sₙ (sum of n terms)
2. Not finding 'd' correctly from word problems
3. Forgetting that AP can have negative common difference
