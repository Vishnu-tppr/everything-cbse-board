# Ch04 — Quadratic Equations ⭐

> **Unit II: Algebra** | **Marks**: 5-8 (every paper) | [[📘 Mathematics]]
> ⭐ **HIGH YIELD** — 5M LA almost guaranteed

## Key Concepts

- Standard form: ax² + bx + c = 0
- **Factorization method** (splitting middle term)
- **Quadratic formula**: x = (-b ± √(b²-4ac)) / 2a
- **Discriminant** D = b² - 4ac:
  - D > 0 → two distinct real roots
  - D = 0 → two equal real roots
  - D < 0 → no real roots
- Nature of roots without solving

## Key Formulas

```
Quadratic Formula: x = (-b ± √D) / 2a where D = b² - 4ac
Sum of roots = -b/a
Product of roots = c/a
If roots are α, β: equation is x² - (α+β)x + αβ = 0
```

## Common Exam Traps ⚠️

1. Calculation errors in discriminant (especially with negatives)
2. Not reducing equation to standard form before applying formula
3. Missing word problem translation
4. Forgetting to write "Hence, the roots are..." at the end
