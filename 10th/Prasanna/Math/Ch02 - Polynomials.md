# Ch02 — Polynomials

> **Unit II: Algebra** | **Marks**: Part of 20 | [[📘 Mathematics]]

## Key Concepts

- Relationship between zeroes and coefficients
- Division algorithm: p(x) = g(x) × q(x) + r(x)
- Finding remaining zeroes when some are given

## Key Formulas

```
Linear: zero = -b/a
Quadratic: sum α + β = -b/a, product αβ = c/a
Cubic: α + β + γ = -b/a, αβ + βγ + γα = c/a, αβγ = -d/a

Quadratic polynomial from zeroes: x² - (α + β)x + αβ
α² + β² = (α + β)² - 2αβ
1/α + 1/β = (α + β)/αβ
```

## Common Exam Traps ⚠️

1. Sign errors in -b/a (forgetting the negative)
2. Not verifying zeroes by substituting back
