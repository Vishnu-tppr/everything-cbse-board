## Key Formulas

```
sin θ = Opposite/Hypotenuse    cosec θ = 1/sin θ
cos θ = Adjacent/Hypotenuse    sec θ = 1/cos θ
tan θ = Opposite/Adjacent      cot θ = 1/tan θ

STANDARD VALUES:
| Angle | sin   | cos   | tan   |
|-------|-------|-------|-------|
| 0°    | 0     | 1     | 0     |
| 30°   | 1/2   | √3/2  | 1/√3  |
| 45°   | 1/√2  | 1/√2  | 1     |
| 60°   | √3/2  | 1/2   | √3    |
| 90°   | 1     | 0     | ∞     |

IDENTITIES:
sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
1 + cot²θ = cosec²θ

COMPLEMENTARY:
sin(90°-θ) = cosθ    cos(90°-θ) = sinθ
tan(90°-θ) = cotθ    sec(90°-θ) = cosecθ
```

## Common Exam Traps ⚠️

1. Not converting all ratios to sin and cos before simplifying
2. Forgetting which identity to use
3. Errors in complementary angle conversions
