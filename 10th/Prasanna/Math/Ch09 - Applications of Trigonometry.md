# Ch09 — Some Applications of Trigonometry

> **Unit V** | **Marks**: Part of 12 | [[📘 Mathematics]]

## Key Concepts

- Angle of elevation and depression
- Height and distance problems
- Problems involving two right triangles

## Key Formulas

```
Height = Distance × tan(angle of elevation)
When two angles given from same point:
  h = d × tan α × tan β / (tan β - tan α)
```

## Common Exam Traps ⚠️

1. Not drawing the figure correctly (angle of depression = angle of elevation at ground)
2. Forgetting to add height of observer
3. Using wrong angle (elevation vs depression)
