# Ch13 — Statistics ⭐

> **Unit VII** | **Marks**: 5-7 (every paper) | [[📘 Mathematics]]
> ⭐ **HIGH YIELD** — 3-5M guaranteed

## Key Formulas

```
MEAN (Direct): x̄ = Σfᵢxᵢ / Σfᵢ
MEAN (Assumed Mean): x̄ = a + Σfᵢdᵢ / Σfᵢ  (dᵢ = xᵢ - a)
MEAN (Step Deviation): x̄ = a + h × (Σfᵢuᵢ / Σfᵢ)  (uᵢ = (xᵢ-a)/h)

MEDIAN: M = l + [(n/2 - cf) / f] × h
  l = lower limit of median class
  cf = cumulative frequency BEFORE median class
  f = frequency of median class
  h = class size

MODE: Mo = l + [(f₁ - f₀) / (2f₁ - f₀ - f₂)] × h
  f₁ = frequency of modal class
  f₀ = frequency of class BEFORE modal class
  f₂ = frequency of class AFTER modal class
```

## Common Exam Traps ⚠️

1. Using wrong cf (must be cumulative frequency BEFORE median class, not including)
2. Modal class identification error
3. Calculation errors in step deviation method
