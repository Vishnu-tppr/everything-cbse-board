## Mendel's Crosses

```
MONOHYBRID (one trait):
  Parents: TT (tall) × tt (short)
  F₁: Tt (all tall)
  F₁ × F₁: Tt × Tt
  F₂: TT : Tt : Tt : tt
  Genotypic ratio: 1:2:1
  Phenotypic ratio: 3:1

DIHYBRID (two traits):
  Parents: RRYY × rryy
  F₁: RrYy (all round yellow)
  F₂: 9:3:3:1
    9 Round Yellow : 3 Round Green : 3 Wrinkled Yellow : 1 Wrinkled Green
```

## Sex Determination

```
Humans: XX-XY system
  Female: XX    Male: XY
  Father determines sex:
    X sperm + X egg → XX (girl)
    Y sperm + X egg → XY (boy)
  Probability: 50% boy, 50% girl
```

## Evolution

- Acquired characters NOT inherited (Lamarck wrong)
- Natural selection (Darwin)
- **Homologous organs**: same structure, different function
- **Analogous organs**: different structure, same function
- Fossils: evidence of evolution

## Common Exam Traps ⚠️

1. Wrong genotypic vs phenotypic ratios
2. Forgetting father determines sex (not mother)
3. Not drawing Punnett square correctly
