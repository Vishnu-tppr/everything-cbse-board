## Nutrition

```
PHOTOSYNTHESIS:
  6CO₂ + 6H₂O →sunlight, chlorophyll→ C₆H₁₂O₆ + 6O₂

HUMAN DIGESTIVE SYSTEM:
  Mouth (salivary amylase, starch→maltose, pH 6.8)
  → Stomach (pepsin, proteins→peptones, HCl, pH 2)
  → Small intestine (bile from liver emulsifies fats)
    (pancreatic: trypsin, lipase, amylase)
    (villi → absorption → large surface area)
  → Large intestine (water absorption)
  → Anus
```

## Respiration

```
AEROBIC: C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O + 38 ATP (mitochondria)
ANAEROBIC (yeast): C₆H₁₂O₆ → 2C₂H₅OH + 2CO₂ + 2 ATP
ANAEROBIC (muscle): C₆H₁₂O₆ → 2C₃H₆O₃ (lactic acid) + 2 ATP
```

## Transportation

```
HEART: 4 chambers, double circulation
  Right atrium → Right ventricle → Lungs (pulmonary)
  Lungs → Left atrium → Left ventricle → Body (systemic)

BLOOD: RBC (O₂), WBC (immunity), Platelets (clotting), Plasma

PLANTS:
  Xylem: water + minerals (upward) — transpiration pull
  Phloem: food (bidirectional) — translocation (uses ATP)
```

## Excretion

```
NEPHRON (functional unit):
  Bowman's capsule → Proximal tubule → Loop of Henle
  → Distal tubule → Collecting duct
  Process: Filtration → Reabsorption → Secretion → Urine
```
