# Ch06 — Control and Coordination

> **Biology** | **Marks**: 3-5 | [[🔬 Science]]

## Nervous System

```
NEURON: Cell body → Dendrite → Axon → Axon terminal → Synapse
REFLEX ARC: Receptor → Sensory nerve → Spinal cord → Motor nerve → Effector

BRAIN:
  Cerebrum: thinking, memory, sensation
  Cerebellum: balance, posture
  Medulla oblongata: breathing, heartbeat
```

## Endocrine Glands

| Gland | Hormone | Function |
|-------|---------|----------|
| Pituitary | Growth hormone | Growth |
| Thyroid | Thyroxine | Metabolism (needs iodine) |
| Pancreas | Insulin | Lowers blood sugar |
| Adrenal | Adrenaline | Fight-or-flight |
| Testes | Testosterone | Male characters |
| Ovaries | Estrogen | Female characters |

## Plant Hormones

| Hormone | Function |
|---------|----------|
| Auxin | Cell elongation, phototropism |
| Gibberellin | Stem elongation, germination |
| Cytokinin | Cell division |
| Abscisic acid | Growth inhibitor |
| Ethylene | Fruit ripening |

## Tropic Movements

- **Photo**tropism: light (shoots +ve, roots -ve)
- **Geo**tropism: gravity (roots +ve, shoots -ve)
- **Hydro**tropism: water (roots +ve)
- **Chemo**tropism: chemicals (pollen tube)
- **Thigmo**tropism: touch (tendrils)
