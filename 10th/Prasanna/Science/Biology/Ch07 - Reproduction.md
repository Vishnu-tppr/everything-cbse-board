# Ch07 — How do Organisms Reproduce?

> **Biology** | **Marks**: 3-5 | [[🔬 Science]]

## Asexual Reproduction

| Method | Organism |
|--------|----------|
| Binary fission | Amoeba |
| Multiple fission | Plasmodium |
| Budding | Hydra, Yeast |
| Spore formation | Rhizopus |
| Fragmentation | Spirogyra |
| Regeneration | Planaria |
| Vegetative | Potato (tuber), Onion (bulb) |

## Sexual Reproduction in Plants

```
FLOWER STRUCTURE:
  Sepals (calyx) → Petals (corolla) → Stamens (anther + filament)
  → Pistil (stigma + style + ovary)

Pollination → Pollen tube → Fertilization → Zygote
Post-fertilization: Ovule → Seed, Ovary → Fruit
```

## Sexual Reproduction in Humans

```
MALE: Testes → Vas deferens → Seminal vesicles → Prostate → Urethra → Penis
FEMALE: Ovaries → Fallopian tubes → Uterus → Vagina

Fertilization in fallopian tube
Implantation in uterus wall
Placenta: nutrition, O₂ to embryo
Menstruation: 28-day cycle
```

## Contraception

- Barrier: condom, diaphragm
- Chemical: oral pills (hormonal)
- IUD: Copper-T
- Surgical: vasectomy (male), tubectomy (female)
