# All Formulas (Math & Physics)

> **Quick Reference Guide** | Memorize these daily | [[📘 Mathematics]] | [[🔬 Science]]

## MATHEMATICS

### Algebra
- **Quadratic Formula**: `x = (-b ± √(b² - 4ac)) / 2a`
- **Discriminant**: `D = b² - 4ac`
- **AP nth term**: `aₙ = a + (n - 1)d`
- **AP Sum**: `Sₙ = n/2 [2a + (n - 1)d] = n/2 [a + l]`

### Coordinate Geometry
- **Distance Formula**: `d = √[(x₂ - x₁)² + (y₂ - y₁)²]`
- **Section Formula**: `(x, y) = ((m₁x₂ + m₂x₁) / (m₁ + m₂), (m₁y₂ + m₂y₁) / (m₁ + m₂))`
- **Midpoint**: `((x₁ + x₂) / 2, (y₁ + y₂) / 2)`

### Trigonometry
- `sin²θ + cos²θ = 1`
- `1 + tan²θ = sec²θ`
- `1 + cot²θ = cosec²θ`
- `sin 30° = 1/2`, `sin 45° = 1/√2`, `sin 60° = √3/2`

### Mensuration (Surface Areas & Volumes)
- **Cylinder**: CSA = `2πrh`, Volume = `πr²h`
- **Cone**: CSA = `πrl`, Volume = `(1/3)πr²h` (where `l = √(r² + h²)`)
- **Sphere**: SA = `4πr²`, Volume = `(4/3)πr³`
- **Hemisphere**: CSA = `2πr²`, TSA = `3πr²`, Volume = `(2/3)πr³`

### Statistics
- **Mean (Direct)**: `x̄ = Σfᵢxᵢ / Σfᵢ`
- **Median**: `M = l + [(n/2 - cf) / f] × h`
- **Mode**: `Mo = l + [(f₁ - f₀) / (2f₁ - f₀ - f₂)] × h`

## PHYSICS

### Light
- **Mirror Formula**: `1/f = 1/v + 1/u`
- **Lens Formula**: `1/f = 1/v - 1/u`
- **Magnification (Mirror)**: `m = -v/u = h'/h`
- **Magnification (Lens)**: `m = v/u = h'/h`
- **Power of Lens**: `P = 1/f` (f in meters, P in Dioptres)
- **Refractive Index**: `n = c / v`

### Electricity
- **Ohm's Law**: `V = IR`
- **Current**: `I = Q/t`
- **Resistivity**: `R = ρ(L/A)`
- **Series Resistance**: `R_eq = R₁ + R₂ + R₃`
- **Parallel Resistance**: `1/R_eq = 1/R₁ + 1/R₂ + 1/R₃`
- **Power**: `P = VI = I²R = V²/R`
- **Joule's Law of Heating**: `H = I²Rt`
