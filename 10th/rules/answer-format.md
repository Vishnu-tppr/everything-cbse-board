# Answer Format Rules

## CBSE Board Answer Structure (ALWAYS enforce)

When helping a student write or improve an answer, ALWAYS use the official CBSE format.

### By Mark Value

```
1-MARK (MCQ):
  → Select correct option
  → For Assertion-Reasoning: evaluate A and R independently

2-MARK (VSA):
  → Line 1: Formula/Definition
  → Line 2: Application/Answer
  → Must include units if numerical

3-MARK (SA):
  → Point 1 with explanation (1M)
  → Point 2 with explanation (1M)
  → Point 3 / Diagram (1M)
  → For numericals: Formula → Substitution → Answer

5-MARK (LA):
  → Given: [all given data]
  → To find/prove: [objective]
  → Solution: [step-by-step with reasons]
  → Therefore: [final answer / Hence Proved]
  → Diagram (if applicable — MUST draw)
  → For theory: 5 well-explained points with examples

4-MARK (Case Study):
  → Read passage → Extract data
  → (i) 1M — usually direct recall
  → (ii) 1M — application
  → (iii) 2M — higher-order thinking / analysis
```

### Subject-Specific Formats

```
MATH: Formula → Substitution → Simplification → Answer (with units)
PHYSICS: Formula → Values → Calculation → Answer (with units) + Diagram
CHEMISTRY: Balanced equation with state symbols + reaction type name
BIOLOGY: Diagram first → Then explanation using biological terms
HISTORY: Date → Event → Cause → Impact (chronological order)
GEOGRAPHY: Location → Conditions → Data/Statistics → Example
POLSCI: Constitutional Article/Amendment → Explanation → Example
ECONOMICS: Data/Statistics → Analysis → Comparison → Conclusion
ENGLISH: Context → Explanation → Textual Quote → Theme connection
```

### Presentation Rules (ALWAYS mention)

1. Underline key terms and final answers
2. Draw diagrams with ruler, label every part
3. Write "Hence proved" / "Therefore" at the end
4. Include units (cm, m, Ω, V, W, J, etc.)
5. Start each new question on fresh space
6. Use Given/To Find/Solution structure for numericals
7. Never use correction fluid — neatly cross out mistakes
